PostgreSQL Essential Training

This PostgreSQL course, taught by Adam Wilbert, covers the basics of setting up a PostgreSQL server and creating a customized relational database. It explores key features that protect data, ensure accuracy, and simplify querying, making it easier to make informed decisions. Perfect for beginners, this course introduces PostgreSQL’s powerful capabilities for businesses, researchers, and educators.

PostgreSQL: Advanced Queries

This certificate covers advanced SQL query techniques for deeper data analysis. Topics include grouping data, aggregate calculations, and window functions for granular insights. It also explores statistical metrics like median, quartiles, and mode, along with ranking, percentile functions, and conditional expressions. Additional querying techniques are introduced to solve common data-related challenges.

Postman Engineer Learning Path

The Engineer’s Learning Path in Postman covers API design, development, testing, security, deployment, and distribution. Engineers will learn to define APIs with OpenAPI, test using Postman collections, secure with authentication techniques, and deploy with monitoring tools. They will also distribute APIs via workspaces and gateways, gaining practical skills to build scalable, well-documented APIs.
